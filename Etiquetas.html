ETIQUETAS MAIN, SECTION Y ARTICLE

A continuación, explicare para que sirven las etiquetas main, section y article.

MAIN

La etiqueta <main> es un contenedor para los temas centrales de un documento o sección. Este contenido central puede interpretarse como todo lo que resta luego de quitar anuncios, encabezados (header), pies (footer), secciones de navegación (nav), vínculos relacionados y otros elementos secundarios.

La etiqueta <main> tiene mayor importancia para personas con discapacidades, que habitualmente acceden a la web a través de programas especiales como los navegadores de voz. En tales casos, los navegadores pueden ser instruidos para ir directamente al contenido principal del documento o sección (main), saltándose toda la información menos relevante.

SECTION

La etiqueta <section> define las secciones de un documento, como capítulos, encabezados, pies de página o cualquier otra sección del documento.
"Una sección es un agrupamiento temático de contenido, normalmente con un encabezado".

ARTICLE

El elemento -article- es un contenedor de bloques de contenido que se consideran independientes del sitio web y pueden, por lo tanto, ser vistos, reutilizados y distribuidos por separado, como por ejemplo, en las indicación. Puedes encontrar habitualmente a este elemento encerrando artículos, entradas de blogs o mensajes de un foro.

ETIQUETAS CODE -PRE

CODE

La etiqueta <code> se utiliza cuando se quiere definir un fragmento o ejemplo de código en línea, sin saltos de línea, escrito en cualquier tipo de lenguaje de programación. Por defecto, <code> se visualiza con un tipo de letra monoespaciado.

Ejemplo:



Pasos para sumar dos numero: 1.num1 2.num2 3.sum = num1 + num2

PRE

La etiqueta -pre- se utiliza cuando queremos definir un texto preformateado en el que se respetan los espacios en blancos, loa tabuladores y los saltos de línea (HTML por defecto no es capaz de hacerlo)

Ejemplo: 



                 Pasos para sumar dos numero: 
                    1. num1 
                    2. num2 
                    3. sum = num1 + num2

ETIQUETA ASIDE

La etiqueta <aside> en HTML se utiliza para marcar contenido relacionado o secundario que está separado del contenido principal de la página, pero que aún guarda cierta relevancia. Por lo general, se coloca en el lado, superior o inferior de una sección principal y puede contener información adicional, como anuncios, enlaces relacionados, widgets, comentarios, etc.
La etiqueta <aside> es útil para estructurar y separar contenido complementario de la información principal de una página web, ayudando a los usuarios a identificar y distinguir fácilmente estos elementos secundarios.

Ejemplo:

Artículo Principal

Solo para amantes del verdadero café.

Anuncio

¡Oferta especial! Compra ahora y obtén un 20% de descuento en nuestros productos. Más información

ETIQUETA U

La etiqueta <u> en HTML se solía utilizar para subrayar texto en un documento web. Sin embargo, en HTML5, su uso se desaconseja en favor de otras técnicas, ya que el subrayado suele asociarse con enlaces en la mayoría de los navegadores web, lo que puede llevar a confusiones.

En lugar de <u>, se recomienda utilizar CSS para aplicar estilos de presentación, como subrayar texto. Puedes lograrlo utilizando propiedades CSS, como text-decoration: underline;, para subrayar texto o elementos específicos de manera más controlada y sin ambigüedades. Esto proporciona un mejor control sobre la apariencia de tu página web y evita malentendidos en la interpretación del subrayado como un enlace.

ETIQUETA SMALL

La etiqueta <small> en HTML se utiliza para representar texto que es de menor importancia o que se debe mostrar en un tamaño de fuente más pequeño en relación con el texto circundante. A menudo se usa para indicar contenido secundario, notas a pie de página, acotaciones o texto legal que debe aparecer en un tamaño de fuente más pequeño y con menos énfasis visual.

El uso de <small> no solo cambia el tamaño del texto, sino que también sugiere visualmente que el contenido es menos prominente en comparación con el texto circundante, lo que puede ayudar a los lectores a identificar información secundaria o menos crítica en una página web.

Ejemplo:


BEBER CON MODERACIÓN. PROHIBIDA SU VENTA A MENORES DE 18 AÑOS

ETIQUETA TABLE

La etiqueta <table> en HTML se utiliza para crear tablas que permiten organizar y mostrar datos en filas y columnas. Las tablas son útiles para representar información en una estructura tabular, como hojas de cálculo, listas de datos, horarios, comparaciones, y más. La etiqueta <table> se utiliza junto con otras etiquetas relacionadas, como <tr> (fila de la tabla), <th> (celda de encabezado) y (celda de datos), para construir la estructura de la tabla.

ETIQUETA TIME

La etiqueta <time> en HTML se utiliza para marcar contenido de texto que representa una fecha y/u hora. Esta etiqueta es útil para ayudar a los motores de búsqueda y los navegadores a entender y presentar correctamente información relacionada con fechas y horas. Puedes proporcionar atributos para indicar la fecha y la hora en un formato específico,

Ejemplo:

La fecha de mi próxima reunión es el 15 de noviembre de 2023 a las 3:00PM


ETIQUETA INS

La etiqueta <ins> se utiliza para marcar texto que ha sido insertado o agregado a un documento, lo que es especialmente útil cuando se está trabajando con documentos versionados o se quiere destacar cambios o adiciones. Por lo general, el texto marcado con ins se muestra subrayado por defecto.

Ejemplo:

El informe anterior fue preparado por el departamento de marketing. Sin embargo, se ha realizado una importante actualización en los datos.


ETIQUETA DEL

La etiqueta <del> se utiliza para marcar el texto que ha sido eliminado o eliminado de un documento. Esto es especialmente útil cuando se trabaja con documentos versionados o cuando se desea resaltar los cambios o eliminaciones realizados en el contenido. Por lo general, el texto marcado con del se muestra con una línea a través por defecto.

Ejemplo:


El informe anterior fue preparado por el departamento de marketing. Sin embargo, se ha realizado una eliminación importante en los datos.


ETIQUETA DATA

La <etiqueta> data en HTML se utiliza para contener valores de datos en un elemento del documento que pueden ser útiles para scripts u otros fines, pero que no se muestran directamente en la página web. Esta etiqueta es especialmente útil cuando deseas asociar datos estructurados con elementos HTML, lo que facilita el acceso a esos datos mediante JavaScript u otras tecnologías.

Ejemplo:

La población de una ciudad es 1,500,000 habitantes.



ETIQUETA SUB

La etiqueta <sub> se utiliza en HTML para crear texto subíndice, lo que significa que el texto dentro de esta etiqueta se muestra ligeramente por debajo de la línea base de texto normal. Esta etiqueta es comúnmente utilizada para representar fórmulas químicas, notaciones matemáticas, números en secuencias y otras situaciones en las que un carácter o conjunto de caracteres debe mostrarse en una posición inferior.

Ejemplo

La fórmula química para el agua es H 2O.

ETIQUETA COLGROUP-THEAD-TBODY-TFOOT

La etiqueta col, <colgroup> en HTML se utiliza para definir un grupo de columnas en una tabla y establecer col que representa la columna. La etiqueta thead se utiliza en HTML para definir la sección de encabezado de una tabla. Esta thead son las etiquetas tr (filas) y th (encabezados de columna). La etiqueta <tfoot> se utiliza en HTML para definir la sección de pie de una tabla. Esta sección generalmente contiene resúmenes, totales u otra información relevante que se coloca al final de una tabla. El contenido dentro de tfoot se coloca después de las filas de datos reales (que suelen estar en tbody) y de las filas de encabezado de columna (que suelen estar en thead).

Ejemplo:

Nombre Edad País
María 25 España
Juan 30 Argentina
Total
